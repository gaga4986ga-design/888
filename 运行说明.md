# 🚀 如何运行这个项目（小白教程）

## 📋 第一步：安装 Node.js

1. **打开浏览器**，访问：https://nodejs.org/
2. **下载 LTS 版本**（绿色按钮，推荐版本）
3. **双击下载的文件**，按照提示安装
4. **安装完成后，重启电脑**（或者关闭所有终端窗口）

## ✅ 第二步：验证安装

1. **打开终端（Terminal）**
   - 在 Mac 上：按 `Command + 空格键`，输入 "终端" 或 "Terminal"，回车
2. **输入以下命令**，按回车：
   ```bash
   node --version
   ```
3. **如果显示版本号**（比如 v20.x.x），说明安装成功 ✅

## 📦 第三步：安装项目依赖

1. **在终端中**，输入以下命令，按回车：
   ```bash
   cd ~/Desktop/888-main
   ```
   （这会进入项目文件夹）

2. **然后输入**，按回车：
   ```bash
   npm install
   ```
   （这会安装所有需要的代码包，可能需要几分钟）

## 🎯 第四步：运行项目

1. **在终端中**，输入以下命令，按回车：
   ```bash
   npm run dev
   ```

2. **等待几秒钟**，你会看到类似这样的信息：
   ```
   ▲ Next.js 15.x.x
   - Local:        http://localhost:3000
   ```

3. **打开浏览器**（Safari、Chrome 等都可以）

4. **在地址栏输入**：`http://localhost:3000`，按回车

5. **🎉 恭喜！** 你应该能看到"天机阁"的页面了！

## 💡 使用提示

- **要停止运行**：在终端中按 `Ctrl + C`（Mac 上是 `Control + C`）
- **要重新运行**：再次输入 `npm run dev`
- **如果遇到错误**：把错误信息复制给我，我来帮你解决

## 🔧 常见问题

### 问题1：提示 "command not found: node"
**解决方法**：说明 Node.js 没有安装成功，请重新执行第一步

### 问题2：提示 "npm install" 很慢或失败
**解决方法**：
- 检查网络连接
- 可以尝试使用国内镜像：
  ```bash
  npm install --registry=https://registry.npmmirror.com
  ```

### 问题3：端口 3000 被占用
**解决方法**：关闭其他可能占用 3000 端口的程序，或者修改端口（告诉我，我来帮你改）
